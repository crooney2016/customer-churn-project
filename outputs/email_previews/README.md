# Email Preview Files

This directory contains static HTML previews of email templates generated by the test suite.

## Generating Previews

To generate all email previews, run:

```bash
python scripts/generate_email_previews.py
```

Or run the test suite:

```bash
pytest tests/test_email_client.py::test_generate_all_previews -v
```

## Preview Files

The following email variations are generated:

### Success Emails

- `success_standard.html` - Standard success email with typical data
- `success_single_risk.html` - Success email with single risk band
- `success_no_risk.html` - Success email with no risk distribution
- `success_large_numbers.html` - Success email with large numbers (testing formatting)
- `success_sorted_risk.html` - Success email showing sorted risk bands

### Failure Emails

- `failure_standard.html` - Standard failure email
- `failure_connection_error.html` - Failure email for connection errors
- `failure_long_traceback.html` - Failure email with long error traceback
- `failure_special_chars.html` - Failure email with special characters in error message

## Viewing Previews

Open any `.html` file in a web browser to preview the email template. These files can also be used to test email client compatibility (Outlook, Gmail, etc.).
